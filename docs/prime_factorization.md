```cpp
template <std::integral T>
std::vector<T> prime_factorization(T n);
```

It returns the prime factors of $n$ in ascending order.

### Constraints
- $n \geq 1$

### Time Complexity
- Supposed to be $O\left(n^\frac{1}{4}\right)$ expected
