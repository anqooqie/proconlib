---
title: $\left\lceil \sqrt{x} \right\rceil$
documentation_of: //tools/ceil_sqrt.hpp
---

```cpp
template <typename T>
T ceil_sqrt(T x);
```

It returns $\left\lceil \sqrt{x} \right\rceil$.

## Constraints
- $0 \leq x \leq 10^{18} + 1000$

## Time Complexity
- $O(\log x)$
